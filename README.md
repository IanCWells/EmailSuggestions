# EmailSuggestions
Chrome Extension for Writing Quick Email Suggestions

Usage:
1. Replace '''const apiKey = "sk-YOURAPIKEY";''' with your API key.

2. In chrome, navigate to chrome://extensions/
   
3. Click on Load Unpacked, and select the EmailSuggestions folder which contains this repos JS code and popup.html

4. When you are in a gmail thread, use the "gather context" button to collect information about the email discussion.
   
5. Click reply, and type '!outline!' to paste in a template.

Templates are written by GPT-3.5-Turbo.  

EmailSuggestionsV1 is the first version of this chrome extension.
